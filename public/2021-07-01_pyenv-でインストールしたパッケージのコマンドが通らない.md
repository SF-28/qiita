## 背景

Windows でも異なるバージョンの Python を扱いたいと思い、 pyenv + Pipenv での仮想環境構築を試してみた

## 各バージョン

- [pyenv](https://github.com/pyenv-win/pyenv-win) : 2.64.8
- [Python](https://www.python.org/downloads/release/python-368/) : 3.6.8
- [Pipenv](https://github.com/pypa/pipenv) : 2021.5.29

## 問題

以下コマンドで Pipenv をインストールしたものの、pipenv コマンドが使えない

```bash
pip install pipenv
```

## 原因

- インストールしたパッケージのコマンドは %PYENV%\shims に配置される
- しかし pip install しただけではコマンドの配置までは実行されない

## 解決策

以下コマンドを実行し %PYENV%\shims も含めて更新する

```bash
pyenv rehash
```

## 余談

- rehash を自動実行してくれるプラグインもある模様
 - [pyenv-pip-rehash](https://github.com/pyenv/pyenv-pip-rehash)
