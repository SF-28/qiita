`:zip`を使う

https://www.erlang.org/doc/man/zip.html

## 圧縮 zip

```elixir
zip_name = ~c'file.zip'
target_list = [~c'./xx/a.txt', ~c'./xx/b.txt'] # ディレクトリも指定可

:zip.create(zip_name, target_list)
```

```elixir
# 圧縮対象のプレフィックスを指定したい場合
zip_name = ~c'file.zip'
target_list = [~c'a.txt', ~c'b.txt']
target_prefix = ~c'./xx' # target_listのプレフィックス

:zip.create(zip_name, target_list, cwd: target_prefix)
```

※distを相対パスで指定した場合、cwdに関係なくカレントディレクトリ基準になる

## 解凍 unzip

```elixir
zip_name = ~c'file.zip'
to_dir = ~c'./' # 解凍先

:zip.extract(zip_name, cwd: to_dir)
```

## 補足

- `:zip`は`:`から始まっていることからも分かる通り、Erlangのモジュール
- そのため、引数は String でなく **CharList** である点に注意
    - `'char_list'`表記は非推奨になる動きがあるので`~c'char_list'`を推奨
    - String から CharList へは`String.to_charlist/1`で変換できる
    - 実は一部の引数以外は String でも動いたが、ややこしいので統一した

## 感想

cwd = CurrentWorkingDirectory と考えると、相対パスの指定が分かりづらい
- `:zip.create` では `圧縮対象リストのプレフィックス`
- `:zip.extract` では `解凍先ディレクトリ`
として捉えるほうが分かりやすいかもしれない
