## 概要

直近に発表されたGitHub Copilotに関するアップデート内容について、要約しました。**新しいAIモデルの追加**、**CLIの機能強化**、**アクセス制御の強化**、**サポートするIDEの拡張** が主な内容です。

## GitHub Copilot CLI: New models, enhanced code search, and better image support

https://github.blog/changelog/2025-11-18-github-copilot-cli-new-models-enhanced-code-search-and-better-image-support/

GitHub Copilot CLIのアップデートにより、最新のAIモデル（GPT-5.1、Gemini 3 Proなど）への対応が追加されました。

また、コード検索機能が強化され、ripgrepのバンドルやgrep、globツールが追加されたことで、より効率的にコードベースを検索できるようになりました。

さらに、画像のサポートも改善され、@メンションだけでなく、貼り付けやドラッグ＆ドロップによる画像追加も可能になり、開発者の生産性向上を目的とした様々な機能強化と信頼性の向上が図られています。

## OpenAI’s GPT-5.1, GPT-5.1-Codex and GPT-5.1-Codex-Mini are now in public preview for GitHub Copilot

https://github.blog/changelog/2025-11-13-openais-gpt-5-1-gpt-5-1-codex-and-gpt-5-1-codex-mini-are-now-in-public-preview-for-github-copilot/

OpenAIの最新モデルである「GPT-5.1」「GPT-5.1-Codex」「GPT-5.1-Codex-Mini」の全スイートが、GitHub Copilotのパブリックプレビューとして展開されました。

これらの新モデルは、Copilot Pro/Pro+/Business/Enterpriseの各プランで利用可能です。

EnterpriseおよびBusinessプランの管理者はCopilot設定でポリシーを有効にする必要があり、Pro/Pro+プランのユーザーはVS Codeのモデルピッカーから有効化できます。

## Gemini 3 Pro is in public preview for GitHub Copilot

https://github.blog/changelog/2025-11-18-gemini-3-pro-is-in-public-preview-for-github-copilot/

Googleの最新モデルであるGemini 3 Proが、GitHub Copilotのパブリックプレビューとして提供開始されました。

これはGitHub Copilot Pro、Pro+、Business、Enterpriseの各サブスクリプションで利用可能になり、段階的に展開されます。

EnterpriseおよびBusinessプランの管理者はCopilot設定でポリシーを有効にする必要があり、Pro/Pro+プランのユーザーはVS Codeのモデルピッカーから有効化できます。

## MCP registry and allowlist controls for VS Code Stable in public preview

https://github.blog/changelog/2025-11-18-internal-mcp-registry-and-allowlist-controls-for-vs-code-stable-in-public-preview/

GitHub Copilotのガバナンス機能として、内部MCP（Microsoft Copilot）レジストリと許可リスト制御がVS Code Stable向けにパブリックプレビューで提供開始されました。

この機能は、Copilot Business/Enterpriseのエンタープライズおよび組織管理者を対象としています。管理者は承認済みのMCPサーバーを登録し、開発者がそのレジストリに登録されたサーバーのみを使用するように強制するポリシーを設定できます。

VS Code Stableでこの統制機能が利用可能になったことで、Copilotユーザーの大半に企業のセキュリティおよび利用ポリシーを適用し、ガバナンスを広く強化することが可能になります。Visual Studioでもサーバー検出がサポートされましたが、制限の強制機能は今後のリリースで提供予定です。

## Fine-grain permissions for Copilot usage metrics now available

https://github.blog/changelog/2025-11-17-fine-grain-permissions-for-copilot-usage-metrics-now-available/

Copilotの利用状況メトリクスへのアクセス権について、より**詳細な権限設定**を導入しました。

これにより、エンタープライズ管理者や請求管理者だけでなく、新設されたエンタープライズロール **View enterprise Copilot metrics** が割り当てられたユーザーも、Copilotの使用状況に関する詳細なAPIおよびダッシュボードにアクセスできるようになります。

## まとめ

2025年11月のGitHub Copilotアップデートは、**開発者の生産性向上**と**組織レベルでのガバナンス強化**の両面で強化されています。CLIの利便性向上や新規モデルの導入は、個々の開発体験を向上させてくれそうです。また、MCPレジストリや詳細なメトリクス権限設定など、企業がCopilotをより安全かつ適切に管理するための基盤も強化されているようです。
